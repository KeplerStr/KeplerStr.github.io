在GitHub上建立仓库

  1. 注册

    GitHub官网

  2. 建立仓库

    • 登录后在左侧找到new新建

    • 按以下步骤创建

在window下安装git

  1. git下载

    下载地址

  2. 安装

    一路next，默认配置就行。

  3. 配置git

    • 在桌面鼠标右键点击Git Bash Here

    • 打开git命令行输入命名ssh-keygen –t rsa –C “邮箱地址”

    • 一路回车，不要输入，就会在默认地址生成密钥文件id_rsa.pub，打开后全选复制到剪贴板

  4. 回到GitHub添加SSH

    • 在头像处下拉setting

    • 添加ssh key

    • 将之前拷贝的密钥拷贝到这里，添加密钥

  5. 验证配置 输入ssh -T git@github.com，这样就配置成功了。

快速使用

配置用户邮箱和密码

  • git config –global user.name “用户名”

  • git config –global user.email “邮箱”

建立本地仓库

  • 在本地建立文件夹，并添加要上传的文件

  • 在这个文件夹使用三个命令进行初始化

    shift+右键进入git命令行，依次执行三个命令，本地生成README文件

上传文件到远程仓库

  • git add命令

    • git add . 上传全部文件
    • git add text.txt 上传指定文件

  • git commit –m ‘会话消息’
