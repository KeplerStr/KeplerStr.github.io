Nginx安装、常用命令以及配置

安装

官网下载：

先安装nginx相关的依赖

  • pcre

  • openssl

  • zlib

  • nginx

安装三步骤

  • 解压 -zxvf -jxvf
  • 配置 执行配置脚本 bash 配置脚本
  • make
  • make install

依次安装上面的依赖库包

最终在/usr/local/nginx/sbin/文件下存在一个nginx脚本可以用来启动nginx

成功启动

在阿里云服务器上添加允许80端口访问，就可以成功访问到上面的页面了。

常用命令

必须进入 /usr./local/nginx/sbin目录下

  • 查看版本号

    ./nginx -v

  • 关闭nginx

    ./nginx -s stop

  • 开启nginx

    ./nginx

  • 查看进程

    ps -ef |grep nginx

  • 重新加载nginx nginx.conf 加载文件

    ./nginx -s reload

配置文件

配置文件的位置

usr/local/nginx/conf/nginx.conf

配置组成

全局块

从配置文件开始到events块之间的内容，主要会设置一些影响 nginx服务器整体运行的配置指令

worker_processes 1; 并发数

evens块

events块涉及的指令主要影响Nginx服务器与用户的网络连接

最大连接数

http块

配置最频繁的部分。

  • http全局块

包含的

  • server块
